# Code of Conduct

## Our Pledge

We want Luminark to be an open, respectful, and constructive project for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and clear in technical discussions
- Assuming good intent while still challenging ideas rigorously
- Giving actionable feedback
- Accepting correction gracefully

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Bad-faith arguments or deliberate disruption
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards and may remove comments, issues, or contributions that violate them.

## Scope

This Code of Conduct applies in project spaces and in public spaces when someone is representing the project.

## Contact

Open a private security report if the issue is security-related. For conduct concerns, contact the maintainers through GitHub.
