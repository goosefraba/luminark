# Luminark v0.1.2

## Changes

- Added Finder/open-file support so packaged app releases can receive markdown files from `Open With` and default-app launches.
- Added packaged markdown document type declarations for `.md`, `.markdown`, `.mdown`, `.mkd`, and `.mkdn`.
- Added release scripts and markdown release-note templates so future GitHub releases render correctly.

## Assets

- Apple Silicon build: `Luminark-0.1.2-macos-arm64.zip`
- Intel build: `Luminark-0.1.2-macos-x86_64.zip`

## Notes

- These app bundles are unsigned and not notarized yet, so macOS may show a security warning on first launch.
- This release targets macOS 15+.
