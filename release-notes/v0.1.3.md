# Luminark v0.1.3

## Changes

- Added clearer README installation guidance for packaged builds
- Added an explicit link to the GitHub Releases page
- Documented which build to use on Apple Silicon vs Intel Macs

## Assets

- Apple Silicon build: `Luminark-0.1.3-macos-arm64.zip`
- Intel build: `Luminark-0.1.3-macos-x86_64.zip`

## Notes

- These app bundles are unsigned and not notarized yet, so macOS may show a security warning on first launch.
- This release targets macOS 15+.
