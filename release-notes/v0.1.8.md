# Luminark v0.1.8

## Fixed

- Fixed a startup race where Finder `Open With` and double-click launches could arrive before SwiftUI had registered its file-open observer, causing the file to be dropped and only the launcher to appear

## Assets

- Apple Silicon build: `Luminark-0.1.8-macos-arm64.zip`
- Intel build: `Luminark-0.1.8-macos-x86_64.zip`

## Notes

- These builds are still unsigned for Developer ID distribution and are not notarized yet
- On first launch, macOS may still require `Open Anyway` or quarantine removal until notarized releases are in place
