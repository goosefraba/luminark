# Luminark v0.1.5

## Added

- New dark metallic cyan `LM` app icon in packaged macOS builds

## Fixed

- The launcher window now comes back to the front when the last viewer window closes
- Packaged release apps now compile and bundle `AppIcon` correctly so the custom app icon shows up end to end

## Assets

- Apple Silicon build: `Luminark-0.1.5-macos-arm64.zip`
- Intel build: `Luminark-0.1.5-macos-x86_64.zip`

## Notes

- These builds are still unsigned for Developer ID distribution and are not notarized yet
- On first launch, macOS may still require `Open Anyway` or quarantine removal until notarized releases are in place
