# Luminark v0.1.1

## Changes

- Added in-app update availability checking
- Added a native `Check for Updates…` command
- Added automatic update checks preference in Settings
- Simplified update flow to open the GitHub release page for manual downloads

## Assets

- Apple Silicon build: `Luminark-0.1.1-macos-arm64.zip`
- Intel build: `Luminark-0.1.1-macos-x86_64.zip`

## Notes

- These app bundles are unsigned and not notarized yet, so macOS may show a security warning on first launch.
- This release targets macOS 15+.
